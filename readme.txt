Easy Make OpenData from WordPress CustomFields.

== Description ==

Making JSON-LD files for your WordPress site, using Custom Fields Content.

Easy Customize "@Context", and easy select output Custom Fields Content.

== Installation ==

1. Upload `opendata-generator` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
